# Сетевые утилиты

Набор скриптов для диагностики и мониторинга сети Windows.

## 📁 Содержание

| Файл | Описание |
|------|----------|
| `опрос_сети.bat` | Диагностический скрипт. Собирает информацию о Wi-Fi, IP-конфигурации, маршрутах, ARP-таблице, проверяет связь с роутером и DNS. Результат сохраняется в network_report.txt. |

## 🚀 Запуск

Просто дважды кликните по файлу или запустите из командной строки:

```cmd
опрос_сети.bat
Что собирает
Шаг	Команда	Информация
1	netsh wlan show interfaces	Wi-Fi подключение, сигнал, скорость
2	ping 8.8.8.8	Доступность интернета
3	ipconfig /all	IP-адреса, шлюз, DNS
4	route print	Таблица маршрутов
5	arp -a	ARP-таблица (соседи по сети)
6	ping 192.168.1.1	Связь с роутером
7	nslookup ya.ru	Работа DNS
🛡️ Безопасность
Скрипт безопасен:

Не требует прав администратора

Не отправляет данные в интернет
